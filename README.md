# seedance-kor-storyforge

한국어 실사용 중심 **Seedance 2.0 프롬프트 엔지니어링 스킬**입니다.  
아이디어만 주면 15초 단일 프롬프트 또는 15초 초과 분할-연장 플랜으로 바로 생성합니다.

---

## 무엇이 다른가

- **한국어 우선**: 실무자가 바로 복붙 가능한 출력
- **질문 최소화**: 정보가 부족해도 초안 먼저 생성
- **실전 안정화**: 메인안 + 안정화안(간소화 버전) 동시 제공
- **플랫폼 리밋 내장**: 첨부 수/길이/용량 초과 방지
- **운영형 확장**: director-mode, multi-agent-plan 반영

---

## 핵심 기능

1. **15초 표준 프롬프트 생성**
   - 타임라인(0-3 / 4-9 / 10-15초) 구조
   - 사운드/BGM/SFX/대사 분리

2. **15초 초과 분할-연장 플랜**
   - SEG1 신규 생성 + SEG2 `@영상1` 연장
   - 엔드프레임 메모 기반 연속성 유지

3. **멀티모달 참조**
   - `@이미지N`, `@영상N`, `@오디오N` 반영

4. **비율 프리셋**
   - 9:16 숏폼 / 16:9 와이드 / 1:1 피드

5. **실패 방지 규칙**
   - 과밀 이벤트/전환 과다/대사 과다 방지

6. **운영 모드**
   - `director-mode`: 목표/제약/승인기준 중심 출력
   - `multi-agent-plan`: 병렬 제작 분할 제안

---

## 플랫폼 제약(Seedance 2.0)

- 이미지: 최대 9장 (`@이미지1~9`), 각 30MB 이하
- 영상: 최대 3개 (`@영상1~3`), 총 2~15초, 각 50MB 이하
- 오디오: 최대 3개 (`@오디오1~3`), 총 15초 이하, 각 15MB 이하
- 총 첨부 수: 최대 12개
- 단일 생성 길이: 4~15초

## 출력 예시(요약)

### A. 15초 이하
- 프로젝트 요약
- Seedance Prompt 타임라인
- 사운드
- 참조
- 네거티브/제한
- 안정화 버전

### B. 15초 초과
- 총괄
- SEG1 신규 생성
- SEG2 연장 생성
- 연결 안정화 규칙
