# seedance-kor-storyforge

한국어 실무형 Seedance 2.0 프롬프트 스킬입니다.  
"아이디어만 주면 바로 제작 가능한 출력"에 집중합니다.

---

## 이 스킬의 목표

- 한국어 사용자에게 즉시 복붙 가능한 결과 제공
- 실패율을 줄이는 구조(안정화/재시도/셀프체크) 내장
- 단일 프롬프트뿐 아니라 15초 초과 분할-연장까지 지원
- 운영형 제작 흐름(director-mode, multi-agent-plan) 지원

---

## 핵심 기능

### 1) 출력 모드 2종
- **director-mode**: 목표/제약/승인 기준 중심
- **executor-mode**: 즉시 제작 가능한 프롬프트 중심

### 2) 버전 출력 표준화
- **A/B/C 3버전** 자동 제공
  - A: 안정형
  - B: 표준형
  - C: 실험형
- 항상 **안정화 버전**(복잡도 축소) 동시 제공

### 3) 실패 대응
- 실패 가능성 높은 경우 **재시도안 자동 출력**
- 기본 전략: 전환/카메라 조합 30% 축소

### 4) 멀티모달 참조 라벨링
- `@이미지N`, `@영상N`, `@오디오N`에 역할 명시
- 예: `@이미지1=캐릭터`, `@영상1=카메라`, `@오디오1=BGM`

### 5) 카메라 레벨 시스템
- Basic(1동작), Pro(2동작), Epic(2~3동작+질감)

### 6) 품질 Self-check 자동 포함
- 번호/역할 매칭
- 파일 제한 준수
- 타임라인 누락 여부
- 금지조건 포함 여부
- 안정화/재시도 포함 여부

---

## 플랫폼 제약(Seedance 2.0)
- 이미지 ≤ 9장, 각 30MB 이하
- 영상 ≤ 3개, 총 2~15초, 각 50MB 이하
- 오디오 ≤ 3개, 총 15초 이하, 각 15MB 이하
- 총 첨부 ≤ 12개
- 단일 생성 길이 4~15초

---

## 추천 사용 흐름

1. 요청 입력(예: "립스틱 광고 9:16")
2. 모드 선택(director/executor)
3. 길이 판단(15초 이하/초과)
4. A/B/C + 안정화 + 재시도안 생성
5. Self-check 확인 후 생성 실행
