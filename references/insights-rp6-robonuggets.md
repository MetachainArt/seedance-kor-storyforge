# insights-rp6-robonuggets

출처: YouTube `rp6_d6sNYXY` 분석 요약(운영 인사이트)

## 핵심 인사이트
- 제작자의 역할을 "프롬프트 노동"에서 "크리에이티브 디렉팅"으로 전환한다.
- 자동화 툴의 복잡한 노드 유지보수 문제를 에이전트 추상화로 줄인다.
- 새 모델 출시 시 문서를 넣어 빠르게 통합하는 운영 유연성이 중요하다.
- 다중 에이전트 병렬 실행으로 처리량을 늘리고, 사람은 승인/품질 판단에 집중한다.
- 공급자 장애/타임아웃을 고려한 fallback 라우팅 규칙이 필요하다.

## 스킬 반영 체크리스트
- director-mode: 목표/제약/승인기준 중심 출력
- multi-agent-plan: 작업 분할표 자동 생성
- fallback-provider: 실패 시 대체 공급자/재시도 문구 자동 제시
- A/B/C안 + 안정화안 동시 출력
- 비용/실패 로그를 노트에 남기는 운영 루틴
