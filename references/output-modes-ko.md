# output-modes-ko

## director-mode 출력 포맷
```text
[디렉팅 브리프]
- 목표:
- 타깃:
- 톤/무드:
- 필수 요소:
- 금지 요소:

[승인 기준]
- 브랜드 일관성:
- 메시지 전달:
- 시각 임팩트:

[제작안]
- A안(안정)
- B안(표준)
- C안(실험)
```
