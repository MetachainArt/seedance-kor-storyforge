# templates-ko

## 1) 광고/제품 템플릿

```text
[프로젝트 요약]
- 목표: 제품 임팩트 광고
- 톤: 프리미엄, 깔끔, 고급 조명
- 길이/비율: 15초 / 9:16

[Seedance Prompt]
0-3초: 어두운 배경에서 제품 실루엣 등장, 림라이트로 윤곽 강조.
4-9초: 제품 180~360도 회전, 재질(금속/유리/패브릭) 반사 디테일 클로즈업.
10-15초: 제품 정면 히어로샷 고정, 로고 영역 비워두고 라이트 페이드아웃.

[사운드]
- BGM: 미니멀 전자음, 저역 강조
- SFX: whoosh, click, subtle sparkle
- 대사: 없음

[네거티브/제한]
- 텍스트 오버레이 없음, 워터마크 없음, 과샤픈 금지
```

## 2) 숏드라마 템플릿

```text
0-4초: 갈등 상황 도입(표정 클로즈업 + 정적 긴장)
5-10초: 감정 충돌(인물 간 거리/시선/대사 1문장)
11-15초: 반전 또는 결심 컷, 다음 화 훅 남기기
```

## 3) 액션/판타지 템플릿

```text
0-3초: 위협 등장, 저각도/핸드헬드 긴장감
4-10초: 핵심 액션 1회 집중(난전 대신 메인 동작 선명화)
11-15초: 여운 컷 + 파티클/잔광 정리
```

## 4) 30초 분할 템플릿

```text
SEG1(0-15): 세계관+갈등 세팅, 마지막 프레임을 명확히 기록
SEG2(15-30): "@영상1을 15초 연장"으로 액션/감정 해소
```
